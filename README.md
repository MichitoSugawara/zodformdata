# bun-ts-package-template

To install dependencies:

```bash
bun install
```

To build:

```bash
bun run build
```

To run:

```bash
bun run src/index.ts
```

To test:

```bash
bun run test
```

To test with update snapshots:

```bash
bun run test:update
```

To test with watch mode:

```bash
bun run test:watch
```
